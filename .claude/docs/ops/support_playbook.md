# Support playbook

> Projet: LeaseAI mobile app MVP complet  
> Positionnement: AI-native IT leasing operator  
> Usage: document source-of-truth pour product, design, mobile, backend, risk, ops et demo.

## Objectif

Fournir une base de support pour partenaires et clients.

## Catégories support

- Problème login.
- Dossier bloqué.
- Pièce manquante.
- Devis incorrect.
- Signature.
- Mandat SEPA.
- Facture.
- Paiement.
- Actif/livraison.
- Renouvellement.

## Priorités

| Priorité | Description | SLA cible MVP |
|---|---|---|
| P0 | Bloque signature/activation | 4h ouvrées |
| P1 | Bloque soumission ou paiement | 1 jour ouvré |
| P2 | Question dossier | 2 jours ouvrés |
| P3 | Demande générale | 3 jours ouvrés |

## Réponses types

### Pièce manquante

> Le dossier est en attente de la pièce suivante: {document}. Ajoutez-la depuis l'onglet Documents pour continuer la revue.

### Signature

> Le contrat est en cours de signature. Dès signature complétée, LeaseAI passera à l'étape d'activation.

### Paiement

> Le paiement du {date} est indiqué comme {status}. Vous pouvez consulter la facture associée dans l'onglet Paiements.
